# FDTE – Residência em IA e Ciência de Dados

Este repositório reúne todo o conteúdo produzido ao longo do Programa de Residência em IA e Ciência de Dados da FDTE, incluindo aulas, exercícios, projetos, avaliações e materiais suplementares.

A organização segue a estrutura dos módulos do curso, permitindo uma navegação clara e progressiva.

## Estrutura

- **01 - Programação para Ciência de Dados (PCD)**
  Conteúdo inicial de Python, manipulação de dados, funções, estruturas, biblioteca padrão e exercícios.

- **02 - Mineração de Dados e Ciência de Dados**
  Conceitos fundamentais, EDA, pré-processamento, normalização, redução de dimensionalidade, preparação de bases.

- **03 - Aprendizado de Máquina (AMQ)**
  Algoritmos supervisionados e não supervisionados, métricas, treinamento, validação, práticas com modelos clássicos.

- **04 - Estatística Aplicada aos Negócios (EAN)**
  Probabilidade, distribuições, testes, estimadores, simulações e fundamentos matemáticos usados em ML.

- **Material Extra**
  Resumos, links, artigos, explicações complementares e materiais de apoio.

## Objetivo

Criar uma base organizada e versionada de todos os materiais estudados, facilitando revisões, estudos e construção de portfólio.

## Tecnologias utilizadas

- Python (pandas, numpy, scikit-learn, matplotlib, seaborn)
- Jupyter Notebook / Google Colab
- Git e GitHub
